# SDK文档

每个队伍的多名AI由同一份代码「分别」操控，不能直接共享信息。

### 常量数值 Constants

- 地图大小、队伍人数等...
- 移速、转速、射速、弹匣容量、炸弹数量
- 伤害数值
- 射程、炸弹爆炸范围
- 传送阵生效所需时间、炸弹爆炸所需时间
- 传送阵、炸弹操作所需时间

### 信息 Observation

- 身份信息。自己的ID与所属队伍。
- 初始地图。含障碍物信息。
- 场上所有玩家的信息。
- 场上所有炸弹的信息。
- 场上所有传送阵的信息；已按传送阵的pattern分类。

### 行动 Action

- 移动。
- 转向。
- 射击。
- 修改传送阵的线段。
- 使用传送阵。
- 设置定时炸弹。

### 辅助工具包 Utils

- 计算坐标间的距离、角度。
- 判断某坐标是否在某传送阵上/某炸弹爆炸范围内。

选手如果有更多需求或建议，可以在交流群中提出。
